### 내장 함수 이용해 숫자 정렬하기 2
**문제**. sorted() 함수를 사용해 숫자 정렬.  
```py
a = [12, 23, 1, 15, 75, 79, 19]
b = sorted(a)
c = sorted(a, reverse=True)
print('a =', a)
print('b =', b)
print('c =', c)

''' result

a = [12, 23, 1, 15, 75, 79, 19]
b = [1, 12, 15, 19, 23, 75, 79]
c = [79, 75, 23, 19, 15, 12, 1]
'''
```
  
- **b = sorted(a)**는 a 리스트의 정렬된 결과를 b로 새로운 리스트를 생성. a 리스트는 그대로 값을 유지.
