### 내장 함수를 이용해 숫자 정렬하기 1
**문제**. 숫자들이 저장되어 있는 리스트를 파이썬 내장 함수를 사용해 정렬하는 코드 작성.
```py
a = [1, 12, 15, 16, 75, 99, 2]
b = [3, -1, 100, 23, 55, 92, 99]

a.sort() # 오름차순 
print(a)

b.sort(reverse=True) # 내림차순순
print(b)

''' result

[1, 2, 12, 15, 16, 75, 99]
[100, 99, 92, 55, 23, 3, -1]
'''
```
  
- 내장 함수 a.sort()를 실행하면 정렬하고자 하는 리스트를 정렬한 후 정렬된 리스트로 기존 리스트가 업데이트.
