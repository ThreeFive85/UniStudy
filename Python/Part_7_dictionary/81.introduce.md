### 자기소개하기
**문제**. 저장되어있는 딕셔너리 자료형을 활용해 출력하는 코드 작성.
```py
# 키 값을 출력하는 첫 번째 방법
a = {'사는 곳': '서울', '성별': '여자', '이름': '시은', '나이': '19', '혈액형': 'b'}
print('내가 사는 곳은', a['사는 곳'])
print('이름은', a['이름'], '성별은', a['성별'])
print('나이는', a['나이'], '혈액형은', a['혈액형'])

print()
# 키 값을 출력하는 두 번째 방법
a = {'사는 곳': '서울', '성별': '여자', '이름': '시은', '나이': '19', '혈액형': 'b'}
print('내가 사는 곳은', a.get('사는 곳'))
print('이름은', a.get('이름'), '성별은', a.get('성별'))
print('나이는', a.get('나이'), '혈액형은', a.get('혈액형'))

''' result

내가 사는 곳은 서울
이름은 시은 성별은 여자
나이는 19 혈액형은 b

내가 사는 곳은 서울
이름은 시은 성별은 여자
나이는 19 혈액형은 b
'''
```
