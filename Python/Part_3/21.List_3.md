### 리스트 추가, 삭제하기
- 리스트의 맨 뒤에 자료(데이터)를 하나 추가할 때는 *append()* 함수를 사용.
- *리스트이름.append(추가하고 싶은 자료)* 형식을 사용.
- 리스트 맨 뒤에 있는 자료를 삭제할 때는 *del 리스트이름[삭제하고 싶은 자료의 위치]* 형식 사용.
- del 명령어를 사용해 하나의 요소를 삭제하면 삭제한 자료 뒤에 있는 자료들은 자동으로 왼쪽으로 붙어 리스트가 완성.
  
```py
a = ['tony', 'park']
a.append('lee')
print(a)
del a[1]
print(a)

''' 실행 결과

['tony', 'park', 'lee']
['tony', 'lee']
'''
```
