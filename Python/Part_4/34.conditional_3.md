### 짝수, 홀수 찾기
**문제**. 정수 세 개를 a, b, c로 입력받아 짝수, 홀수를 구분해 출력하는 코드 작성.
```py
a = int(input('첫 번째 정수 : '))
b = int(input('두 번째 정수 : '))
c = int(input('세 번째 정수 : '))

if a % 2 == 1:
    print(a, '홀수')
else:
    print(a, '짝수')

if b % 2 == 1:
    print(b, '홀수')
else:
    print(b, '짝수')

if c % 2 == 1:
    print(c, '홀수')
else:
    print(c, '짝수')

첫 번째 정수 : 1
두 번째 정수 : 2
세 번째 정수 : 3
1 홀수
2 짝수
3 홀수
'''
```
