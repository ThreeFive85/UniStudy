### 선택 제어문

#### 예제 4-1. 입력된 정수의 짝수와 홀수 판별 프로그램

```c
#include <stdio.h>
void main() {
    int a;

    printf("정수 입력: ");
    scanf("%d", &a);

    if(a % 2 == 0)
        printf("%d 은(는) 짝수. \n", a); // 나머지가 0인가를 검증
    if(a % 2 == 1)
        printf("%d 은(는) 홀수. \n", a); // 나머지가 1인가를 검증
}

/* 실행 결과

정수 입력: 10
10 은(는) 짝수.
*/
```

#### 예제 4-2. 입력된 점수에 대한 학점 평가 프로그램

```c
#include <stdio.h>

void main() {
    int score;

    printf("점수를 입력하시오 : ");
    scanf("%d", &score);

    if(score >= 90)
        printf("%d A \n", score);
    else if(score >= 80)
        printf("%d B \n", score);
    else if(score >= 70)
        printf("%d C \n", score);
    else if(score >= 60)
        printf("%d D \n", score);
    else
        printf("%d F \n", score);
}

/* 실행 결과

점수를 입력하시오 : 90
90 A
*/
```

#### 예제 4-3. 계산기 프로그램

```c
#include <stdio.h>

void main() {
    int op_num1, op_num2;
    char op;

    printf("계산기 프로그램\n");
    printf("연산자 입력 (+, -, *, /) : ");
    scanf("%c", &op);

    printf("%c의 피연산자 1 입력 : ", op);
    scanf("%d", &op_num1);
    printf("%c의 피연산자 2 입력 : ", op);
    scanf("%d", &op_num2);

    switch(op) {
        case '+' :
            printf("%d + %d = %d\n", op_num1, op_num2, op_num1 + op_num2);
            break;
        case '-' :
            printf("%d - %d = %d\n", op_num1, op_num2, op_num1 - op_num2);
            break;
        case '*' :
            printf("%d * %d = %d\n", op_num1, op_num2, op_num1 * op_num2);
            break;
        case '/' :
            printf("%d / %d = %d\n", op_num1, op_num2, op_num1 / op_num2);
            break;
        default:
            printf("잘못된 연산자입니다.\n");
    }
}

/* 실행 결과

계산기 프로그램
연산자 입력 (+, -, *, /) : *
*의 피연산자 1 입력 : 10
*의 피연산자 2 입력 : 55
10 * 55 = 550
*/
```
