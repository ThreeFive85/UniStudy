### 선택 제어문

#### 예제 4-1. 입력된 정수의 짝수와 홀수 판별 프로그램

```c
#include <stdio.h>
void main() {
    int a;

    printf("정수 입력: ");
    scanf("%d", &a);

    if(a % 2 == 0)
        printf("%d 은(는) 짝수. \n", a); // 나머지가 0인가를 검증
    if(a % 2 == 1)
        printf("%d 은(는) 홀수. \n", a); // 나머지가 1인가를 검증
}

/* 실행 결과

정수 입력: 10
10 은(는) 짝수.
*/
```

- 선택 제어문의 종류
  - if문
  - switch ~ case문
  - goto문
