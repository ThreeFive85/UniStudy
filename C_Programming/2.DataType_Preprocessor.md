### 상수와 변수의 개념

#### 예제 2-1. 상수, 변수의 정의와 선언에 관련한 프로그램

```c
#include <stdio.h>
#define PI 3.142592 // 매크로 정의를 이용한 상수 정의
const int K = 10; // const를 이용한 상수 정의

void main() {
    double a; // 변수 a를 선언
    a = K + PI;
    // K = 20; 상수는 변하는 값이 아니므로 값을 변화시키면 에러 발생
    printf("%f\n", a);

    int x, y, z; // 변수 x, y, z를 선언

    x = 10, y = 20; // 변수 x, y에 값을 할당
    z = x + y;
    printf("%d\n", z);
}
```

- 변수의 특징
  - 모든 변수는 이름을 갖는다.
  - 모든 변수는 정해진 자료형을 갖는다.
  - 모든 변수는 할당된 값을 갖는다(초기화).
